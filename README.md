# hashtag-generator

### Installation
- install and configure Go 1.6+
- `go get github.com/bruschill/hashtag-generator`

### Setup
- copy `$PROJECT_ROOT/.env.example` to `$PROJECT_ROOT/.env`

### Usage
- From the root of the project directory:
  - `go build`
  - `./hashtag-generator`
- visit `localhost:9000` in your browser
